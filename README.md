# capstone-coursera
## Little Lemon Restaurant — Table Booking Web App

A React web application built as the capstone project for the Meta Front-End Developer course on Coursera.

## Features
- Home page with hero, specials, testimonials and about sections
- Table reservation form with validation
- Booking confirmation page
- Responsive design (mobile & desktop)
- Dark mode toggle
- Accessible markup (ARIA labels, semantic HTML)
- Unit tests with Jest and React Testing Library

## Tech Stack
- React
- React Router DOM
- CSS (custom, no frameworks)
- Jest + React Testing Library

## Getting Started

### Installation
```bash
npm install
npx create-react-app my-app
npm install react-router-dom
```

### Run the app
```bash
npm start
```

### Run tests
```bash
npm run test
```
